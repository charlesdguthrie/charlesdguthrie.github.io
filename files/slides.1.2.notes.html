Import Matplotlib¶

Before we do anything, import matplotlib's pyplot module and run matplotlib inline. The inline command is a Jupyter Notebook command make charts visible in our notebook, as opposed to creating an image file elsewhere:

In [34]:
import matplotlib.pyplot as plt
%matplotlib inline

Figures and Axes¶

To draw any plot, we create a Figure object, then add to it one or more Axes objects. There are shortcuts to this approach, but we will start with the fundamental one.

A Figure is the canvas on which to draw a chart, which is called a 'subplot' or Axes object. Don't confuse Axes with the two Axis objects, the x-axis and y-axis. The image below, from matplotlib's user docs, illustrates the distinction.

In [35]:

Below, we create a Figure object and an Axes object. In the add_subplot function we must specify three parameters to indicate how many plots we are drawing, and which one we are selecting. add_subplot(1,1,1) draws a 1x1 grid of plots and selects the 1st one.

In [36]:
#create figure
fig = plt.figure()

#draw 1 plot; 'ax' will be the 1st of 1.
ax = fig.add_subplot(1,1,1) #(num. rows, num. cols, id for this chart)

There is a lot of flexibility here and you can make many charts per figure, but for the beginning of this course we will just focus on one chart at a time.

In [37]:
#create figure
fig2 = plt.figure()

#create a 2x2 grid of subplots
ax1 = fig2.add_subplot(2,2,1) #ax1 will be the 1st
ax2 = fig2.add_subplot(2,2,2) #ax2 will be the 2nd plot
ax3 = fig2.add_subplot(2,2,3) #3rd plot
ax4 = fig2.add_subplot(2,2,4) #4th plot

#Adding text to clarify which is which
ax1.text(0.5,0.5, '1',ha='center',va='center',size=20,alpha=.5)
ax2.text(0.5,0.5, '2',ha='center',va='center',size=20,alpha=.5)
ax3.text(0.5,0.5, '3',ha='center',va='center',size=20,alpha=.5)
ax4.text(0.5,0.5, '4',ha='center',va='center',size=20,alpha=.5)

First Bar Chart¶

Let's put something on the plot!

Let's say we are plotting the heights of people in a family, with the following heights:

  • Mom: 5 feet
  • Dad: 6 feet
  • Sis: 4 feet
  • Bro: 3 feet
  • Junior: 1 foot

First we create data:

  • x_locations = locations of the bars on the x-axis.
  • bar_heights = height of each bar in sequence.
In [38]:
x_locations = [1,2,3,4,5]
bar_heights = [5,6,4,3,1]

Like before, we create a Figure and add an Axes, but this time we draw a bar chart.

The first arguments to the bar method are locations of the bars on the x axis, and the heights of the bars.

In [39]:
#create figure
fig = plt.figure()

#create subplot
ax = fig.add_subplot(1,1,1)

#add bars
ax.bar(x_locations,bar_heights)

<Container object of 5 artists>
